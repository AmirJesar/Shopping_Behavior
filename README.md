# Shopping_Behavior
# Sales Profit Prediction Project

## Overview
This project predicts the profit generated by selling the most popular products over the year. Insights are provided by categories such as Gender, Age, and Season.

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the project: `python app.py`

## Features
- Profit prediction
- Analysis by Gender, Age, Season
- Visualizations and reports

## Technologies
- Python
- Pandas, Matplotlib, Scikit-learn, Streamlit, Plotly
